hexo模板假设在github上的总结

1.下载nodejs

2.墙内的话利用npm安装一个cnpm

npm install -g cnpm –registry=http://registry.npm.taobao.org

3.安装hexo

cnpm install -g hexo-cli

4.hexo的常用命令

hexo n；hexo s；hexo；d；hexo clean；hexo help

5.把localhost上调试好的blog推到github上

本来以为挺简单的可能半个小时就搞定了，但是没想到遇到了各种教程上未提及的错误，花了好几个小时，但是又多会了一些东西，希望下次可以做的更好。
