基本数据类型和String类型的转换

  1. 将基本数据类型转换为String，+ ""即可，相当于基本数据类型和空字符串拼接（我自己说的不确定有没有理解错）
    例：
    String str1 = 1 + ""; //以类型为String的str1接收"1"

  2. 将String类型转换成基本数据类型，通过基本类型包装类调用parseXX方法。包装类型会在面向对象部分详细讲
    例：
    int i = Integer.parseInt("123"); //将"123"转换为int类型的123，用i接收

  3. 把String转换成char，这话的意思是把字符串的第一个字符存储到char中，因为String是字符串有多个字符
    例：
    char c = ("123".charAt(0)); //String的charAt方法

基本数据类型和String类型的转换细节

  1. 将String类型转换成基本数据类型时，要确保String类型能够转换成有效数据。
    比如”Hello”转换成基本类型，用int接收的话就会抛出异常。

  2. 如果格式不正确，会抛出异常造成程序终止。（编译不会报错，而是运行时出异常）
