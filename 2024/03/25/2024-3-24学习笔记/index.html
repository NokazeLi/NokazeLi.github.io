二维数组

动态初始化2(先声明后创建)：

二维数组种的一维数组，长度可以不相同，比如{ {1},{1,2},{1,2,3} }

静态初始化：

int[][] arr = {{1,2,3},{2,3,3},{100}}; 这就是

用二维数组表示输出杨辉三角

杨辉三角特点归纳：

  1. 第1行有1个元素，第n行有n割元素。

  2. 每一行的第一个和最后一个元素都为1。

  3. arr[i][j] = arr[i - 1][j] + arr[i - 1][j - 1]即除了首尾之外，本行的第j个元素等于上一行的j元素与j-1元素之和。

二维数组的细节和注意事项；

一维数组的声明方式有int[] x;或者int x[];而二维数组的声明方式有int[][] y; int y[][]; int[] y[]三种。

类和对象(OOP)

如前所述，程序是一个世界，用对象代表这个世界中的各种事务(属性、行为)。

创建一个对象；实例化一个对象是一个意思，都是创建一个类的具体对象。

类是抽象的，代表数据类型。对象是具体的实例

对象在内存中的存在形式

  1. 成员变量 = 属性 = field

  2. 属性是类的一个组成部分，一般是基本数据类型，也可以是用用类型(对象，数组)。

属性的注意事项和语法

  1. 属性的定义语法和变量相同
    例：
    访问修饰符(public,protected,private) 属性类型 属性名;

  2. 属性的定义类型可以为任意类型，包含基本类型和引用类型。

  3. 属性如果不赋值有默认值，规则和Array相同。

类的创建(内存变化)和数组差不多

类和对象的内存分配机制

内存结构分析

  1. 栈：一般存放基本数据类型(局部变量)

  2. 堆：存放对象或数组

  3. 方法区

方法的调用机制原理:

栈里的代码执行完毕后会退出相应的栈空间。

成员方法的定义

访问修饰符 返回数据类型 方法名(形参列表){
方法体；
return；
}

访问修饰符：控制方法的适用范围。

返回数据类型：

  1. 一个方法最多有一个返回值(如果需要多个返回值，可以返回数组)

  2. 返回类型可以是任意类型(基础类型和引用类型)

  3. 如果方法要求有返回数据类型，则方法体中最后执行的语句必须是return，而且要求返回值类型必须和return的值类型一致或者兼容。

  4. 如果是void，则可以没有return语句或者return语句只含有return。

成员方法定义注意事项和细节

  1. 一个方法可以有0个参数也可以有多个参数，多个参数中间用逗号隔开。

  2. 参数类型可以为基本类型和引用类型。

  3. 调用带参数的方法时，一定要传入相同类型或者兼容的类型。

  4. 方法定义时的参数称为形式参数(形参)；方法调用时的参数称为实际参数(实参)。形参和实参的类型要一致或者兼容，个数和顺序必须一致。

不可以在方法中 再定义另一个方法。

方法调用的细节

  1. 在同一个类中的方法调用:直接调用即可。

  2. 跨类调用时，要先创建对象。

方法的传参机制(极其重要)

  1. 对于基本类型,传递的时值(值copy)，形参的任何改变不影响实参。

  2. 对于引用类型则是地址copy，形参的改变同样影响实参。

写方法的思路

  1. 方法的返回类型

  2. 方法的名字

  3. 形参

  4. 方法体

以上四步依序确认。
