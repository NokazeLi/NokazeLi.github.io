2024-3-01学习笔记

今天是基本类型的强制转换、String类型和基本类型之间的转换、还有运算符的算数运算符部分的内容。

基本数据类型强制转换

是自动类型转换的逆过程，将容量大的数据类型转换为容量小的数据类型，需要加上强制转换符（xx），但是可能会造成精度降低或溢出，应注意。

例：

int n1 = (1.9);
System.out.println(n1); //输出1，精度损失

int n2 = 2000;
byte b1 = (byte)n2;
System.out.println(b1); //输出-48，数据溢出了，并不是输出byte的上限

强制类型转换细节

  1. 当精度大的数据转换成精度小的数据，就是强制转换

  2. 强转符号只针对于最近的操作数有效，往往会使用小括号提升优先级
    例：
    int x = (int)10 * 3 + 5 * 1.5; //只会让10强转为int，整个运算结果还是double类型，会编译报错

int y = (int)(10 * 3 + 5 * 1.5);
System.out.println(b1); //double运算结果44.0强转为int，精度损失，输出44

  1. char类型可以保存int的常量值，但是不能保存int的变量值，需要强转。
    例：
    char c1 = 100; //输出d
    int m = 100;
    char c2 = m; //int类无法自动转换成char，编译报错
    char c3 = (int)n; //正常输出d

基本数据类型和String类型的转换

  1. 将基本数据类型转换为String，+ ""即可，相当于基本数据类型和空字符串拼接（我自己说的不确定有没有理解错）
    例：
    String str1 = 1 + ""; //以类型为String的str1接收"1"

  2. 将String类型转换成基本数据类型，通过基本类型包装类调用parseXX方法。包装类型会在面向对象部分详细讲
    例：
    int i = Integer.parseInt("123"); //将"123"转换为int类型的123，用i接收

  3. 把String转换成char，这话的意思是把字符串的第一个字符存储到char中，因为String是字符串有多个字符
    例：
    char c = ("123".charAt(0)); //String的charAt方法

基本数据类型和String类型的转换细节

  1. 将String类型转换成基本数据类型时，要确保String类型能够转换成有效数据。
    比如”Hello”转换成基本类型，用int接收的话就会抛出异常。

  2. 如果格式不正确，会抛出异常造成程序终止。（编译不会报错，而是运行时出异常）

运算符

一种用以表示数据的运算、赋值和比较等的特殊符号。有算数运算符、赋值运算符、关系运算符[比较运算符]、逻辑运算符、位运算符（需要二进制基础）和三元运算符。

算数运算符————对数值类型的变量进行运算

着重说明\、%、++

\的应用

System.out.println(10 \ 4); //在Java中因为是两个int型之间的运算，所以运算结果也是int型，相比于数学运算结果的浮点数2.5丢失了精度，所以输出2。

System.out.println(10.0 \ 4); //10,0是double，结果是double，2.5

double d = 10 \ 4; //因为在java中10\4是2，而因为是double类型变成2.0
`System.out.println(d); //所以输出2.0

++

如果独立使用的话i++和++i相同

int i = 10;
i++; //本质是i = i + 1;
System.out.println(i); //输出11
++i;
System.out.println(i); //输出12

但当作为表达式使用时，++i是先自增后赋值，i++是先赋值后自增
例：
int j = 8;
int k = ++j; //等价j = j + 1; k = j; 所以k = 9 j = 9
int h = j++; //等价 h = j; j = j + 1;因为保留了上面的数值 所以h = 9，j = 10

一些练习；
int i = 1; //1赋值给i
i = i++; //(1)temp = i;(2)i = i + 1;(3)i = temp; 这里是重点，涉及自动临时变量的问题,temp就是临时变量
System.out.println(i); //所以会输出1

int i = 1; //1赋值给i
i = ++i；//(1)i = i + 1;(2)temp = i;(3)i = temp;同样使用临时变量temp
System.out.println(i); //所以会输出2

2024-2-29学习笔记

28号学习了基础类型中的数值类，今天学习了剩下的。

字符类型 char

char可以存储单个字符

做个字符用String类存储

char有两个字节可以存储单个汉字

字符类型细节

  1. 字符常量用单引号括起来，不可以用双引号 双引号括起来的是字符串

  2. char可以储存转义字符

  3. 在Java中char的本质是一个整数，输出的是Unicode码对应的字符

  4. 给char类型赋一个数值，会输出对应的Unicode码代表的字符
    例：
    char c1 =97；
    System.out.println(c1); //输出a

    也可以反向输出，但是要用强转类型

    char c2 =’a’;
    System.out.println(c1); //输出97

  5. 因为char的本质是一个整数，所以可以参与计算
    例：
    System.out.println(‘a’ + 10); //输出97

具体数值代表的字符可以查询ASCII表

字符类型的本质

  1. 字符类型存储到计算机中，需要把字符对应的码值找出来
    比如：
    存储：’a’ ==> 码值97 ==> 二进制 ==> 存储
    读取: 二进制 ==> 97 ==> ‘a’ ==> 显示

  2. 字符和码值的顺序是通过字符编码表规定的

常用的字符编码

ASCII表；1个字节表示，一共有128个字符（实际一个字节至多可以表示256个字符）ASCII是上世纪六十年代美国指定的一套字符编码，对英语字符与二进制位之间的关系做了统一规定，共规定了128个字符的编码，值占用了1个字节的后七位，最前面一位统一规定为0
缺点：只能表示英文字母

Unicode：Unicode编码表，固定大小的编码，字母和汉字统一占用2个字符。Unicode是一种将世界上所有符号都纳入其中的编码，每一个符号都给予了一个独一无二的编码，使用Unicode编码不存在乱码问题.
缺点:单个英文字母和汉字都占用2个字节，比较浪费存储空间。
Unicode编码的0-127与ASCII相同，所以Unicode兼容ASCII

UTF-8：大小可变的编码，字母使用1个字节而汉字使用3个字节，所以UTF-8能存储比gbk更多的汉字.UTF-8是互联网上使用最广的一种Unicode的实现方式。是一种变长的编码方式，它可以使用1-6字节表示一个符号，会根据不同的符号而变化字节长度。

gbk:可以表示汉字而且范围广，字母使用1个字节汉字使用2个字节存储

gb2312:可以表示汉字，表示的汉字较少 gb2312 < gbk

big5:繁体中文

布尔型

  1. boolean只允许取值true和false，不能是null。

  2. boolean只占1个字符。

  3. 在Java里不可以用0和非0代替false和true。

基本数据类型自动转换

当Java程序在进行赋值或者运算时，精度小的类型自动转换为精度大的数据类型，这就是自动类型转换

数据类型按精度（容量）大小排序 很重要，需要背过

char → int → long → float → double

byte → short → int → long → float → double

自动数据类型转换细节

  1. 有多种类型的数据混合运算时，系统首先自动将所有数据类型转换成容量最大的那种数据类型，然后再进行运算。

  2. 当我们把精度（容量）大的数据类型赋值给精度（容量）小的数据类型会报错，反之则自动进行类型转换

  3. byte和short不会与char之间自动相互转换。

当把一个数值赋值给byte类时，先判断该数是否在byte范围内，若是则没问题。（这里不符合整数常量默认int类型规则）
例：
byte b1 = 10; //正常编译不报错

int n1 = 10;
byte b1 = n2; //报错，因为要将int类数值存储进精度较小的byte中

  1. byte、short、char三者可以运算，在运算时首先转换为int类型。
    例；
    byte b2 = 1;
    byte b3 = 2;
    short s1 = 1;
    byte b4 = b2 + s1; //如细节4所述，先转换成int再运算，byte接收int数据不会自动转换，编译报错

    byte b4 = b2 + b3; //两个同为byte类运算，也需要先自动转换为int类，所以用byte接收也会编译报错

  2. boolean类不参与自动转换

Ctrl+Shift+D 复制光标所在的整行，并粘贴到下一行。

Ctrl+Shift+F 删除光标所在的整行。

2024-2-28学习笔记

今天主要学习了变量的部分知识

变量

为什么需要变量

因为一个程序就是一个世界，要用不同的变量代表这个世界中不同的数据

简单说明一下程序运行时，电脑和内存的状态

int a = 1; //先在内存中开辟一个储存空间，储存的数值是1类型是int，名字是a作为地址，相当于房间的门牌号
a = 3; //找到a地址，并由a指向储存数值的空间，将空间内储存的类型为int数值为1的值，改为3
System.out.println(a); //经由a指向找到储存空间并输出数值3
上例主要为了说明修改变量的数值，并不会开辟新的存储空间，而是改变旧空间中的数值。

变量使用的基本步骤

1.声明变量
int a;
2.赋值
a = 60;
3.调用

当然也可以声明+赋值一步到位int a = 60;

变量使用的注意事项

1.变量表示内存中的一个存储空间（不同的变量，类型不同，占用的空间大小也不同）

2.该区域有自己的名称[变量名]和类型[数据类型]

3.变量必须先声明后使用

4.该区域数据可以在同一类型的范围内不断变化（强类型特性）

5.变量在同一作用域内不可以重名

6.变量有三个要素，类型、名称、值

程序中加号的使用

1.左右两边都是数值类型时，则作加法运算

2.左右两边有一方是字符串时，则做拼接运算

数据类型

整数型

每一种数据都定义了明确的数据类型（故Java是强类型语言），在内存中分类了不同大小的内存空间

基本数据类型分为数值型、字符型、布尔型

数值型又分为整数类和浮点类

整数类：byte[1字节]、short[2字节]、int[4字节]、long[8字节]

浮点类：float[4字节]（单精度）、double[8字节]（双精度）

字符型:char[2字节] //用于存放单个字符，比如’a’

布尔型:boolean[1字节]//存放true、false

整数型 细节

1.Java个整数类型有固定的范围和字段长度，不受OS（Operation System）[操作系统]
以保证Java程序的可移植性

2.Java的整型常量默认为int型，声明long型常量后需加L或者l

3.Java程序中变量常声明为int型，除非不足以表示大数才使用long

4.bit（位）；计算机中的最小储存单位
byte（字节）；计算机中的基本存储单元
1byte = 8bit

浮点型

1.与整数类相似，Java浮点类型也有固定的范围和字段长度，不受OS影响

2.Java浮点型常量默认为double型，声明float型需要在后面加F或者f

3.浮点型常有两种表示形式
十进制数的形式，如 5.12，512.0f，.512(必须有小数点)
科学计数法形式如5.12e2[5.12*10的2次方]，5.12E-2[5.12/10的2次方]

4.通常情况下，应该使用double型，因为精确度更高。即保存尾数更多

5.浮点数陷阱
double a = 2.7;
double b = 8.1 / 3
上述两个数值并不相等，所以切记不要直接对运算过后结果位小数的值做相等判断

比如

if(a == b){
System.out.println("相等");
}

比较好的方法是

if(Math.abs(a - b) < 0.00001)}{
System.out.println("相等");
}

Math.abs()是绝对值

Java API（Application Programming Interface）即应用程序编程接口

Java类的组织形式（和使用API文档相关）

API文档的使用方法

JDK由很多的包组成

每个包又有很多的接口、类、枚举和异常组成。

每个类下面又有字段、构造器（构造方法）和成员方法

比如要找ArrayList

1.当知道目标类在哪个包的时候，比如ArrayList在java.util包中，直接找到java.util包然后找到ArrayList类，然后找具体方法找如何使用S。

2.不知道包的时候 ，点开API文档，点索引输入类名，然后选中类点显示。

2024-2-27学习笔记

今天主要是一些Dos相关的代码和注释之类的问题

常用转义符

\t制表符，用于实现对其功能，视觉效果字符中间加空格

\n换行符，顾名思义换行

\\为了实际输出一个\

\"为了实际输出一个双引号”

\r输出一个回车，实际作用光标回到本行首位，后续的字符会覆盖前面已有的 实际应用还不熟悉

注释

1.单行注释//

2.多行注释/**/

3文本注释：可以通过javadoc输出的各种，如@author @version详情可以看书。

Java代码规范 （部分）

1.类和方法的注释要以Javadoc的方式来写（即文档注释）

2.非Javadoc的注释往往是给代码维护者看的，应着重说明为什么这样写、如何修改、有什么注意事项等

3.要移动代码的话，先整体选中然后按tab右移或者shift+tab左移

4.运算符和等号的左右两边各加一个空格

5.源文件使用UTF-8代码

6.行宽不要超过80个字符

7.代码风格有两种次行风格与行尾风格
行尾风格特点:大括号左端在行尾 推荐
次行风格特点:大括号左端另起一行 这种用的人少，但也是可以的

Java_Dos原理

Dos(Disk Operation System)磁盘操作系统

1.接收指令
2.解析指令
3.执行指令

在变成领域文件夹也被称为目录

相对路径:从当前目录开始定位而形成的路径

绝对路径:从顶级目录（如C盘D盘等）开始定位而形成的路径

一些常用命令

dir查看当前目录的内容列表，也可后续加[路径]查看

cd切换盘符，例如cd /D c:当前路径由D盘切换至C盘
cd ..切换至上一级目录
cd \直接切换至根目录

cls清屏

exit退出Dos

额外带一点

在Dos中输入几个字母，点tab可以实现命令补全

写好的.java文件在cmd中用javac [文件名.java]编译，然后用java [类名]运行。

一点Java历史

Java于1995年问世，最初由SUN公司运营现在由Oracle公司运营。14年的8和18年的11是比较有代表性的版本。

Java的重要特点

1.面向对象（oop）暂时没学到

2.健壮性、强类型语言、异常处理、垃圾自动收集 暂时没学到

3.跨平台性（因为jvm），编译过后的.class可以多平台运行（还是因为jvm）

4.Java是解释性语言

编程语言分为解释性语言（Js、PHP、Java）和编译性语言（C、C++）

解释性语言编译后的代码不能直接被机器执行，需要解释器

编译性语言编译后的代码就是二进制了，可以被机器执行

常用Java开发软件

editplus、notepad++、sublime text、IDEA、eclipse，后两者功能强大实际工作中被广泛应用。

Java核心机制————JVM(Java Virtual Machine)

1..JVM 具有指令集并使用不同的存储区域。负责执行命令，管理数据、内存、寄存器，包含在JDK中

2.不同的平台有不同的jvm，如WindowsJVM、LinuxJVM

JDK(Java Development Kit)简介

1.JDK=JRE（java runtime environment） +java开发工具（Java、javac、javadoc、javap）

2.JRE = JVM+ Java核心类库（类）

*** 如果想要运行一个开发好的Java程序，安装JRE即可 ***
