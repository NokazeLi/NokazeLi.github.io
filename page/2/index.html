2024-2-28学习笔记

今天主要学习了变量的部分知识

变量

为什么需要变量

因为一个程序就是一个世界，要用不同的变量代表这个世界中不同的数据

简单说明一下程序运行时，电脑和内存的状态

int a = 1; //先在内存中开辟一个储存空间，储存的数值是1类型是int，名字是a作为地址，相当于房间的门牌号
a = 3; //找到a地址，并由a指向储存数值的空间，将空间内储存的类型为int数值为1的值，改为3
System.out.println(a); //经由a指向找到储存空间并输出数值3
上例主要为了说明修改变量的数值，并不会开辟新的存储空间，而是改变旧空间中的数值。

变量使用的基本步骤

1.声明变量
int a;
2.赋值
a = 60;
3.调用

当然也可以声明+赋值一步到位int a = 60;

变量使用的注意事项

1.变量表示内存中的一个存储空间（不同的变量，类型不同，占用的空间大小也不同）

2.该区域有自己的名称[变量名]和类型[数据类型]

3.变量必须先声明后使用

4.该区域数据可以在同一类型的范围内不断变化（强类型特性）

5.变量在同一作用域内不可以重名

6.变量有三个要素，类型、名称、值

程序中加号的使用

1.左右两边都是数值类型时，则作加法运算

2.左右两边有一方是字符串时，则做拼接运算

数据类型

整数型

每一种数据都定义了明确的数据类型（故Java是强类型语言），在内存中分类了不同大小的内存空间

基本数据类型分为数值型、字符型、布尔型

数值型又分为整数类和浮点类

整数类：byte[1字节]、short[2字节]、int[4字节]、long[8字节]

浮点类：float[4字节]（单精度）、double[8字节]（双精度）

字符型:char[2字节] //用于存放单个字符，比如’a’

布尔型:boolean[1字节]//存放true、false

整数型 细节

1.Java个整数类型有固定的范围和字段长度，不受OS（Operation System）[操作系统]
以保证Java程序的可移植性

2.Java的整型常量默认为int型，声明long型常量后需加L或者l

3.Java程序中变量常声明为int型，除非不足以表示大数才使用long

4.bit（位）；计算机中的最小储存单位
byte（字节）；计算机中的基本存储单元
1byte = 8bit

浮点型

1.与整数类相似，Java浮点类型也有固定的范围和字段长度，不受OS影响

2.Java浮点型常量默认为double型，声明float型需要在后面加F或者f

3.浮点型常有两种表示形式
十进制数的形式，如 5.12，512.0f，.512(必须有小数点)
科学计数法形式如5.12e2[5.12*10的2次方]，5.12E-2[5.12/10的2次方]

4.通常情况下，应该使用double型，因为精确度更高。即保存尾数更多

5.浮点数陷阱
double a = 2.7;
double b = 8.1 / 3
上述两个数值并不相等，所以切记不要直接对运算过后结果位小数的值做相等判断

比如

if(a == b){
System.out.println("相等");
}

比较好的方法是

if(Math.abs(a - b) < 0.00001)}{
System.out.println("相等");
}

Math.abs()是绝对值

Java API（Application Programming Interface）即应用程序编程接口

Java类的组织形式（和使用API文档相关）

API文档的使用方法

JDK由很多的包组成

每个包又有很多的接口、类、枚举和异常组成。

每个类下面又有字段、构造器（构造方法）和成员方法

比如要找ArrayList

1.当知道目标类在哪个包的时候，比如ArrayList在java.util包中，直接找到java.util包然后找到ArrayList类，然后找具体方法找如何使用S。

2.不知道包的时候 ，点开API文档，点索引输入类名，然后选中类点显示。

2024-2-27学习笔记

今天主要是一些Dos相关的代码和注释之类的问题

常用转义符

\t制表符，用于实现对其功能，视觉效果字符中间加空格

\n换行符，顾名思义换行

\\为了实际输出一个\

\"为了实际输出一个双引号”

\r输出一个回车，实际作用光标回到本行首位，后续的字符会覆盖前面已有的 实际应用还不熟悉

注释

1.单行注释//

2.多行注释/**/

3文本注释：可以通过javadoc输出的各种，如@author @version详情可以看书。

Java代码规范 （部分）

1.类和方法的注释要以Javadoc的方式来写（即文档注释）

2.非Javadoc的注释往往是给代码维护者看的，应着重说明为什么这样写、如何修改、有什么注意事项等

3.要移动代码的话，先整体选中然后按tab右移或者shift+tab左移

4.运算符和等号的左右两边各加一个空格

5.源文件使用UTF-8代码

6.行宽不要超过80个字符

7.代码风格有两种次行风格与行尾风格
行尾风格特点:大括号左端在行尾 推荐
次行风格特点:大括号左端另起一行 这种用的人少，但也是可以的

Java_Dos原理

Dos(Disk Operation System)磁盘操作系统

1.接收指令
2.解析指令
3.执行指令

在变成领域文件夹也被称为目录

相对路径:从当前目录开始定位而形成的路径

绝对路径:从顶级目录（如C盘D盘等）开始定位而形成的路径

一些常用命令

dir查看当前目录的内容列表，也可后续加[路径]查看

cd切换盘符，例如cd /D c:当前路径由D盘切换至C盘
cd ..切换至上一级目录
cd \直接切换至根目录

cls清屏

exit退出Dos

额外带一点

在Dos中输入几个字母，点tab可以实现命令补全

2024-2-26学习笔记

啃了几天书，发现太晦涩了而且自己的基础也不够强，进行不下去。今天开始换老韩的教程，重新开始巩固JavaSE。

简单的Java程序

因为之前直接用IDEA入门的，所以对编译和运行有点一知半解。这次终于弄清楚了

写好的.java文件在cmd中用javac [文件名.java]编译，然后用java [类名]运行。

一点Java历史

Java于1995年问世，最初由SUN公司运营现在由Oracle公司运营。14年的8和18年的11是比较有代表性的版本。

Java的重要特点

1.面向对象（oop）暂时没学到

2.健壮性、强类型语言、异常处理、垃圾自动收集 暂时没学到

3.跨平台性（因为jvm），编译过后的.class可以多平台运行（还是因为jvm）

4.Java是解释性语言

编程语言分为解释性语言（Js、PHP、Java）和编译性语言（C、C++）

解释性语言编译后的代码不能直接被机器执行，需要解释器

编译性语言编译后的代码就是二进制了，可以被机器执行

常用Java开发软件

editplus、notepad++、sublime text、IDEA、eclipse，后两者功能强大实际工作中被广泛应用。

Java核心机制————JVM(Java Virtual Machine)

1..JVM 具有指令集并使用不同的存储区域。负责执行命令，管理数据、内存、寄存器，包含在JDK中

2.不同的平台有不同的jvm，如WindowsJVM、LinuxJVM

JDK(Java Development Kit)简介

1.JDK=JRE（java runtime environment） +java开发工具（Java、javac、javadoc、javap）

2.JRE = JVM+ Java核心类库（类）

*** 如果想要运行一个开发好的Java程序，安装JRE即可 ***

JDK配置(设置环境变量)

因为执行的程序 如果在当前目录下不存在，则系统会在一个名为path的环境变量指定目录中查找，若无法找到则报错
