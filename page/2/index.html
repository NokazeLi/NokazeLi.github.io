循环引申的编程思想

switch分支结构

  1. switch关键字，表示switch分支。

  2. 表达式对应一个值。

  3. 当表达式的值等于常量1时，就执行语句块1。

  4. break;表示退出switch代码块。

  5. 如果case 常量1不匹配就继续向下匹配case 常量2，以此类推。

  6. 如果都没匹配成功，则执行default。

没有break;代码会向下穿透

switch分支细节

  1. 表达式数据类型应和case后的常量类型一致，或者是可以自动转换成可以相互比较的类型，比如输入的是char，而常量是int。

  2. switch的表达式的返回值必须是：byte、short、int、char、enum、String。

  3. case后面跟的必须是常量或者常量表达式，不能是变量。

  4. default子句是可选的。

switch和if的选择

  1. 如果判断的具体数值不多，而且符合byte、short、int、char、enum、String类型，虽然两种都能用，但是推荐使用switch。

  2. 对区间判断，对结果为boolean类型的判断，用if-else较好。

位运算符

二进制（Binary）在运算中的说明

  1. 二进制是逢2进位制，0、1是基本算符。

  2. 现代的电子计算机技术全部采用的二进制，因为只使用0、1两个算符，非常简洁方便，易于用电子方式实现。

原码、反码、补码（重点 要背过）

  1. 二进制的最高位是符号位：0表示正数，1表示负数。

  2. 正数的原码、反码、补码都一样。（三码合一）

  3. 负数的反码 = 原码符号位不变，其他位取反。

  4. 负数的补码 = 反码 + 1，负数的反码 = 补码 - 1。

  5. 0的反码和补码都是0。

  6. Java没有无符号数，换言之Java中得数都是有符号的。

  7. 计算机运算得时候都是以补码得方式进行的。（所以自己算要先反 → 补 → 算 → 反 → 原）

  8. 看运算结果，要看原码。

位运算符

Java中有七个位运算符（&、|、^、~、>>、<<、>>>）

第一组：

3. 按位异或(^)；两位一个为0一个为1时，则结果为1，否则为0.

  1. 按位取反(~)： 0 -> 1, 1 -> 0

  1. 算术右移(>>)：低位溢出（就是舍弃的意思），符号位不变，并用符号位补齐溢出的高位。

标识符命名规则与规范

  1. Java对各种变量、方法和类的命名时使用的字符序列称为标识符

  2. 凡是自己可以起名字的地方都叫标识符

命名规则（若违反则编译报错）

  1. 由26个英文字母大小写，0-9、$和_组成。

  2. 数字不能作为开头

  3. 不可以使用关键字和保留字，但是可以包含关键字和保留字。

  4. Java中严格区分大小写，长度无限制。

  5. 不能包含空格

命名规范（若违背不会编译报错，但是应当遵守）

  1. 包名：多单词组成时所有字母都小写,例 aaa.bbb.ccc

  2. 类名、接口名：由多个字母组成时所有单词的首字母都大写，例：AaaBbbCbb

  3. 变量名、方法名：由多单词组成时，第一个单词首字母小写，从第二个单词开始首字母大写，例：aaaBbbCcc

  4. 常量名；所有字母都大写，多单词时每个单词之间用_连接，例：AAA_BBB_CCC

保留字：现在的Java版本中尚未使用，但是以后版本可能会作为关键字使用。

    规则：将该数不断的除以16，直到商除不尽为止，然后将每步得到的余数倒过来就是对应的二进制。（16转10的逆过程）

关系运算符（比较运算符）

关系运算符的结果是boolean类型，只能是true和false

关系表达式经常用在if结构的条件或者循环结构的条件中

==、!=、<、>、<=、>=、instanceof(检查是否是类的对象，到面向对象时会细讲)

关系运算符组成的表达式，称为关系表达式

逻辑运算符

用于连接多的条件（多个关系表达式），最终结果也是一个boolean值

&、&&、|、||、!、^（逻辑异或）

逻辑运算符规则
a & b:&叫逻辑与 a和b同时为true则结果为true否则为false。
a && b：短路与 a和b同时为true则结果为true否则为false。

a | b：逻辑或 当a和b中有一个为true则结果为true否则为false。
a || b：短路或 当a和b中有一个为true则结果为true否则为false。
!a:取反或者叫非运算。当a为true时结果为false，反之亦然。

a ^ b：逻辑异或，当a和b不同时结果为true否则为false。

&和&&的使用区别

  1. &&：如果第一个条件为false，则最终结果直接为false第二个条件不会判断，效率高。

  2. &：不管第一个条件是否为false，第二个都要判断，效率低。

  3. 开发中基本都是使用&&

|和||的使用区别

  1. ||：如果第一个条件为true，则最终结果直接为true第二个条件不会判断，效率高。

  2. |：不管第一个条件是否为true，第二个都要判断，效率低。

  3. 开发中基本都是使用||。

逻辑非与逻辑异或

!a:逻辑非或者逻辑取反，若a为true则!a为false，反之亦然。

a^b：逻辑异或，a和b的结果不同时，a^b为true；a和b结果相同时a^b为false。

一个练习(读代码分析过程和输出结果)

因为过程会有点长，在下面分析：
if((z++ == 46)&&(y = true))z++;
//z++ == 46,先比较后自增，所以为true，z为47；因为时&&程序进关系表达式2，y = true 注意这里是=赋值不是==，所以也是true。条件符合进入代码体 z++，z为48
if((x = false)||(++z == 49))z++;
//同样第一部分是赋值=不是==，x为false因为是||所以要判断条件2，++z == 49先自增后比较z为49且条件为true，进入代码体z++，z为50.
所以最终输出为50。

字符类型 char

char可以存储单个字符

字符类型细节

  1. 字符常量用单引号括起来，不可以用双引号 双引号括起来的是字符串

  2. char可以储存转义字符

  3. 在Java中char的本质是一个整数，输出的是Unicode码对应的字符

  4. 给char类型赋一个数值，会输出对应的Unicode码代表的字符
    例：
    char c1 =97；
    System.out.println(c1); //输出a

    也可以反向输出，但是要用强转类型

    char c2 =’a’;
    System.out.println(c1); //输出97

  5. 因为char的本质是一个整数，所以可以参与计算
    例：
    System.out.println(‘a’ + 10); //输出97

具体数值代表的字符可以查询ASCII表

字符类型的本质

  1. 字符类型存储到计算机中，需要把字符对应的码值找出来
    比如：
    存储：’a’ ==> 码值97 ==> 二进制 ==> 存储
    读取: 二进制 ==> 97 ==> ‘a’ ==> 显示

  2. 字符和码值的顺序是通过字符编码表规定的

常用的字符编码

ASCII表；1个字节表示，一共有128个字符（实际一个字节至多可以表示256个字符）ASCII是上世纪六十年代美国指定的一套字符编码，对英语字符与二进制位之间的关系做了统一规定，共规定了128个字符的编码，值占用了1个字节的后七位，最前面一位统一规定为0
缺点：只能表示英文字母

Unicode：Unicode编码表，固定大小的编码，字母和汉字统一占用2个字符。Unicode是一种将世界上所有符号都纳入其中的编码，每一个符号都给予了一个独一无二的编码，使用Unicode编码不存在乱码问题.
缺点:单个英文字母和汉字都占用2个字节，比较浪费存储空间。
Unicode编码的0-127与ASCII相同，所以Unicode兼容ASCII

UTF-8：大小可变的编码，字母使用1个字节而汉字使用3个字节，所以UTF-8能存储比gbk更多的汉字.UTF-8是互联网上使用最广的一种Unicode的实现方式。是一种变长的编码方式，它可以使用1-6字节表示一个符号，会根据不同的符号而变化字节长度。

gbk:可以表示汉字而且范围广，字母使用1个字节汉字使用2个字节存储

gb2312:可以表示汉字，表示的汉字较少 gb2312 < gbk

big5:繁体中文

布尔型

  1. boolean只允许取值true和false，不能是null。

  2. boolean只占1个字符。

  3. 在Java里不可以用0和非0代替false和true。

基本数据类型自动转换

当Java程序在进行赋值或者运算时，精度小的类型自动转换为精度大的数据类型，这就是自动类型转换

数据类型按精度（容量）大小排序 很重要，需要背过

char → int → long → float → double

byte → short → int → long → float → double

自动数据类型转换细节

  1. 有多种类型的数据混合运算时，系统首先自动将所有数据类型转换成容量最大的那种数据类型，然后再进行运算。

  2. 当我们把精度（容量）大的数据类型赋值给精度（容量）小的数据类型会报错，反之则自动进行类型转换

  3. byte和short不会与char之间自动相互转换。

当把一个数值赋值给byte类时，先判断该数是否在byte范围内，若是则没问题。（这里不符合整数常量默认int类型规则）
例：
byte b1 = 10; //正常编译不报错

int n1 = 10;
byte b1 = n2; //报错，因为要将int类数值存储进精度较小的byte中

  1. byte、short、char三者可以运算，在运算时首先转换为int类型。
    例；
    byte b2 = 1;
    byte b3 = 2;
    short s1 = 1;
    byte b4 = b2 + s1; //如细节4所述，先转换成int再运算，byte接收int数据不会自动转换，编译报错

    byte b4 = b2 + b3; //两个同为byte类运算，也需要先自动转换为int类，所以用byte接收也会编译报错

  2. boolean类不参与自动转换

2024-2-28学习笔记

今天主要学习了变量的部分知识

变量

为什么需要变量

因为一个程序就是一个世界，要用不同的变量代表这个世界中不同的数据

简单说明一下程序运行时，电脑和内存的状态

int a = 1; //先在内存中开辟一个储存空间，储存的数值是1类型是int，名字是a作为地址，相当于房间的门牌号
a = 3; //找到a地址，并由a指向储存数值的空间，将空间内储存的类型为int数值为1的值，改为3
System.out.println(a); //经由a指向找到储存空间并输出数值3
上例主要为了说明修改变量的数值，并不会开辟新的存储空间，而是改变旧空间中的数值。

变量使用的基本步骤

1.声明变量
int a;
2.赋值
a = 60;
3.调用

当然也可以声明+赋值一步到位int a = 60;

变量使用的注意事项

1.变量表示内存中的一个存储空间（不同的变量，类型不同，占用的空间大小也不同）

2.该区域有自己的名称[变量名]和类型[数据类型]

3.变量必须先声明后使用

4.该区域数据可以在同一类型的范围内不断变化（强类型特性）

5.变量在同一作用域内不可以重名

6.变量有三个要素，类型、名称、值

程序中加号的使用

1.左右两边都是数值类型时，则作加法运算

2.左右两边有一方是字符串时，则做拼接运算

数据类型

整数型

每一种数据都定义了明确的数据类型（故Java是强类型语言），在内存中分类了不同大小的内存空间

基本数据类型分为数值型、字符型、布尔型

数值型又分为整数类和浮点类

整数类：byte[1字节]、short[2字节]、int[4字节]、long[8字节]

浮点类：float[4字节]（单精度）、double[8字节]（双精度）

字符型:char[2字节] //用于存放单个字符，比如’a’

布尔型:boolean[1字节]//存放true、false

整数型 细节

1.Java个整数类型有固定的范围和字段长度，不受OS（Operation System）[操作系统]
以保证Java程序的可移植性

2.Java的整型常量默认为int型，声明long型常量后需加L或者l

3.Java程序中变量常声明为int型，除非不足以表示大数才使用long

4.bit（位）；计算机中的最小储存单位
byte（字节）；计算机中的基本存储单元
1byte = 8bit

浮点型

1.与整数类相似，Java浮点类型也有固定的范围和字段长度，不受OS影响

2.Java浮点型常量默认为double型，声明float型需要在后面加F或者f

3.浮点型常有两种表示形式
十进制数的形式，如 5.12，512.0f，.512(必须有小数点)
科学计数法形式如5.12e2[5.12*10的2次方]，5.12E-2[5.12/10的2次方]

4.通常情况下，应该使用double型，因为精确度更高。即保存尾数更多

5.浮点数陷阱
double a = 2.7;
double b = 8.1 / 3
上述两个数值并不相等，所以切记不要直接对运算过后结果位小数的值做相等判断

比如

if(a == b){
System.out.println("相等");
}

比较好的方法是

if(Math.abs(a - b) < 0.00001)}{
System.out.println("相等");
}

Math.abs()是绝对值
